Module bittongue::grapheme[−][src]

Expand description

This module exports utilites related to grapheme clusters/unicode segmentation.

Some “visual” characteres are actually multiple unicode characters, such as letter “a” with three diacritics “ā̤́”, those are grapheme clusters.
